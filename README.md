# aBetterMe — Brand Board

A single-file brand board for **aBetterMe**: identity, palette, typography, and messaging in one HTML page, plus the exported WhatsApp branding asset pack.

## View it

Download `index.html` and open it in any browser. No install, no build step, no server.

## What's inside

- `index.html` — the complete brand board
- `aBetterMe WhatsApp branding.zip` — exported WhatsApp branding assets
